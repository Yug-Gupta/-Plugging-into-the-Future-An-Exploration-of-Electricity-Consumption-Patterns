
# Energy Consumption: An Exploration of Electricity Patterns

## About The Project

This project was created to demonstrate a complete data visualization workflow. It takes a raw electricity consumption dataset, cleans and processes it using Python's Pandas library, and serves the final visualizations—a Tableau dashboard and story—through a lightweight Flask web application.

The goal is to provide a clear, end-to-end example of how to prepare data for analysis and present it professionally on the web.

### Built With

*   [Python](https://www.python.org/) - Data processing and backend
*   [Pandas](https://pandas.pydata.org/) - Data cleaning
*   [Flask](https://flask.palletsprojects.com/) - Web framework
*   [Tableau Public](https://public.tableau.com/) - Data visualization

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

*   Python 3.7+
*   pip (Python package installer)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd <repository-folder>
    ```

2.  **Create and activate a virtual environment (recommended):**
    *   On Windows:
        ```bash
        python -m venv venv
        .\venv\Scripts\activate
        ```
    *   On macOS/Linux:
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```

3.  **Install the required packages:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Add the data:**
    *   Create a `data` folder in the project's root directory.
    *   Place your raw data file, named `Consumption (1).csv.xls`, inside the `data` folder.

## Usage

### 1. Prepare the Data

Run the data preparation script. This will read the raw CSV, clean it, and save a new file ready for Tableau.

```bash
python load_data.py
```

This will create `cleaned_consumption_data.csv` inside an `output` folder.

### 2. Create and Embed Tableau Dashboard & Story

1.  Go to Tableau Public.
2.  Upload the `output/cleaned_consumption_data.csv` file as a new data source.
3.  Create your worksheet and dashboard.
4.  Once your dashboard is saved, click the "Share" button on your visualization.
5.  Copy the "Embed Code".
6.  Open the `templates/index.html` file in this project and paste the embed code into the corresponding section in the `<body>`.

### 3. Run the Web Application

Start the Flask server:

```bash
python app.py
```

### 4. View the Project

Open your web browser and navigate to:

```
http://127.0.0.1:5000
```

You should see your Flask application displaying the embedded Tableau dashboard.
